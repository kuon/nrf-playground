## Nrf dev playground

This repository is a playground for the nrf52840 chip.

### Requirements

- `rust` >= `nightly 1.48.0`
- `probe-run` <https://ferrous-systems.com/blog/defmt/>

Rustup target:

`rustup target add thumbv7em-none-eabihf`
